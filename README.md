# loginApp

### Getting Started

Clone the repository in your machine. After Installing all the dependencies, run the back-end **server** file and front-end **mobile** file by locating folders simultaneously
using the command; ``` npm start ```
  
### Testing Application

The app can be run on android emulator, IOS simulator or Expo Go (can be downloaded from IOS store or PlayStore)
